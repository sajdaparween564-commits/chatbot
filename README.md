# Campus Buddy Chatbot

The Campus Buddy chatbot is designed to assist students in navigating campus life by providing information about events, classes, and resources.

## Features
- Provides real-time information about campus events and activities.
- Offers class schedules and details.
- Answers common student inquiries.
- Integrates with campus resources and facilities.

## Installation Instructions
1. Clone the repository:
   ```bash
   git clone https://github.com/sajdaparween564-commits/chatbot.git
   cd chatbot
   ```
2. Install dependencies:
   ```bash
   npm install
   ```

## Deployment Guide
To deploy the Campus Buddy chatbot:

### Using Heroku
1. Create a Heroku account and install Heroku CLI.
2. Login to your Heroku account:
   ```bash
   heroku login
   ```
3. Create a new Heroku app:
   ```bash
   heroku create <app-name>
   ```
4. Push your code to Heroku:
   ```bash
   git push heroku main
   ```

## Usage Examples
- To ask about campus events, type:
  ```
  What events are happening this week?
  ```
- To get class schedules, type:
  ```
  Show me my class schedule for Monday.
  ```
